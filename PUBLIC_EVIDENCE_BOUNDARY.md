# Public Evidence Boundary

This repository is intentionally designed as a public evidence portfolio.

It demonstrates professional research engineering, reproducibility, validation discipline, and macro-financial modelling structure without exposing proprietary ShockBridge Pulse logic or private scientific methodology.

- Clean Python project organization
- Reproducible research structure
- Financial time-series handling
- Volatility-regime feature engineering
- Safe dependence-mapping demonstrations
- Machine-learning workflow design
- Walk-forward validation discipline
- Backtesting architecture
- Report-generation capability
- Documentation quality

## This Repository Does Not Reveal

- Private ShockBridge Pulse scoring logic
- Proprietary transmission-chain weights
- Commercial signal thresholds
- Private client-facing report engine

- Full MTV-GARCH architecture
- Advanced Airy / GMW wavelet implementation

All methods implemented here must be simplified, educational, or demonstration-level.
