# Research Question

How can macro-financial shocks be studied with reproducible Python pipelines, volatility-regime features, time-aware validation, and decision-oriented reporting?

This repository focuses on the public-safe engineering layer of macro-financial research:

- data ingestion
- feature engineering
- volatility regimes
- dependence mapping
- machine-learning classification
- validation
- backtesting
- reporting

The goal is to demonstrate professional research execution without exposing proprietary ShockBridge Pulse methodology.
