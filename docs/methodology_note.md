# Methodology Note

This public repository uses simplified and transparent research components to demonstrate implementation ability.

The methodology is intentionally public-safe. It does not contain private ShockBridge Pulse scoring logic, proprietary weighting systems, or advanced unpublished research methods.

## Public Demonstration Components

- Rolling return features
- Rolling volatility features
- Simplified dependence measures
- Basic regime labels
- Machine-learning classification skeleton
- Walk-forward validation structure
- Backtest framework
- Research report generation

## Private Methodology Boundary

Advanced models related to dynamic canonical dependence, proprietary volatility-transmission scoring, advanced wavelet structures, and commercial report intelligence remain outside this repository.
