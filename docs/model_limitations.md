# Model Limitations

This repository is a public evidence portfolio, not a production trading system.

## Limitations

- Public models are simplified.
- Outputs are for research demonstration only.
- No investment advice is provided.
- Proprietary ShockBridge Pulse scoring logic is excluded.
- Advanced unpublished scientific methodology is excluded.
- Backtests are illustrative unless otherwise stated.
- Transaction costs, liquidity, slippage, and execution risks must be considered before any practical use.

## Purpose

The purpose is to prove professional research engineering, validation discipline, and communication quality.
