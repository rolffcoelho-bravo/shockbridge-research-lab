# Public/Private Boundary

This repository is intentionally sanitized.

## Public-safe

This repository may include:

- high-level methodology summaries
- public research framing
- non-proprietary data categories
- reproducibility philosophy
- sanitized diagrams or conceptual examples
- public-facing research notes

## Private-only

The following must remain private:

- exact thresholds
- exact weights
- exact signal rules
- full methodology scripts
- raw validation internals
- ML fold details
- private hedge-review logic
- research-desk action policy internals
- private lock files
- raw private data panels

## Rule

The public repository should demonstrate research credibility without exposing the private engine.
